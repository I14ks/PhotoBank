# Документация Фотобанка

Этот каталог содержит документацию для проекта **Фотобанк**.

## Структура

```
docs/
├── developer/          # Документация разработчика
│   ├── intro.md
│   ├── architecture/   # Архитектура системы
│   ├── docgen/         # Генерация документации
│   ├── api/            # API Reference
│   └── deployment/     # Развёртывание
├── user/               # Документация пользователя
│   ├── wiki/           # Wiki-статьи
│   ├── faq/            # Частые вопросы
│   └── guides/         # Руководства
└── assets/             # Изображения для документации
```

## Запуск документации

### Установка зависимостей

```bash
npm install
```

### Запуск dev-сервера

```bash
npm run start
```

Откройте `http://localhost:3000`

### Сборка production-версии

```bash
npm run build
npm run serve
```

## Вклад в документацию

1. Создайте новую ветку
2. Добавьте/измените `.md` файлы в `docs/`
3. Создайте Pull Request

## Стиль документации

### Документация разработчика

- Технические детали
- Примеры кода
- API спецификации

### Документация пользователя (Wiki)

- Структура в стиле MediaWiki/FANDOM
- Внутренние ссылки через `[[wiki/page|текст]]`
- Содержание с якорями
- Футер с категоризацией
