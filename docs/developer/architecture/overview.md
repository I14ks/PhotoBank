# Архитектура системы

## Обзор

Фотобанк построен по архитектуре **клиент-сервер** с разделением на frontend и backend.

```
┌─────────────────┐         ┌─────────────────┐
│   Frontend      │         │    Backend      │
│   (React/TS)    │◄───────►│   (FastAPI)     │
│                 │  HTTP   │                 │
└─────────────────┘  JSON   └─────────────────┘
                                  │
                                  ▼
                          ┌─────────────────┐
                          │   In-Memory     │
                          │   Database      │
                          └─────────────────┘
```

## Компоненты

### Frontend

- **React 18** — библиотека для построения UI
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **Motion** — анимации
- **Lucide React** — иконки

### Backend

- **FastAPI** — веб-фреймворк
- **Pydantic** — валидация данных
- **PyJWT** — работа с JWT токенами
- **Hashlib** — хеширование паролей

## Структура проекта

```
PhotoBank/
├── backend/
│   ├── main.py           # Точка входа API
│   └── requirements.txt  # Python зависимости
├── src/
│   ├── App.tsx           # Главный компонент
│   ├── main.tsx          # Точка входа React
│   └── index.css         # Глобальные стили
├── docs/
│   ├── developer/        # Документация разработчика
│   └── user/             # Документация пользователя
└── docusaurus.config.js  # Конфигурация документации
```
