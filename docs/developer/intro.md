# Введение в документацию разработчика

Добро пожаловать в документацию разработчика проекта **Фотобанк**.

## О проекте

**Фотобанк** — это интерактивная веб-платформа для визуального поиска, хранения и обмена графическим контентом. Система поддерживает загрузку, редактирование и умный поиск изображений.

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Backend | Python 3.11+, FastAPI |
| Frontend | React 18+, TypeScript |
| База данных | In-memory (для демонстрации) |
| Аутентификация | JWT (JSON Web Tokens) |
| Стилизация | Tailwind CSS |

## Быстрый старт

### Предварительные требования

- Python 3.11 или выше
- Node.js 18 или выше
- npm или pnpm

### Запуск backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Запуск frontend

```bash
npm install
npm run dev
```

## Соглашения о стиле кода

### Python (Backend)

В проекте используется следующий стиль кодирования:

- **PEP 8** — базовый стандарт оформления кода
- **Длина строки** — 79 символов
- **Имена переменных** — однобуквенные (в соответствии с внутренним стилем)
- **Комментарии** — только docstrings, без комментариев в теле функций

### TypeScript/React (Frontend)

- **TypeScript** — строгая типизация
- **React** — функциональные компоненты с хуками
- **Стилизация** — Tailwind CSS с утилитарными классами

Документация генерируется автоматически из:
