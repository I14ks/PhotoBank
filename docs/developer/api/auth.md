# Auth Module (Сгенерировано)

> **Примечание:** Эта документация сгенерирована автоматически из `auth_module.py`.

---

## Обзор

Модуль аутентификации предоставляет функции для управления пользователями.

---

## Константы

| Константа | Значение | Описание |
|-----------|----------|----------|
| `SALT` | `"photobank-salt-2026"` | Соль для хеширования |
| `SECRET` | `"your-secret-key..."` | Ключ для JWT |
| `ALGO` | `"HS256"` | Алгоритм JWT |
| `EXPIRE` | `30` | Срок действия токена (мин) |

---

## Функции

### `h(p: str) -> str`

Хеширует пароль с солью используя SHA-256.

**Параметры:**
- `p` (str): Исходный пароль пользователя

**Возвращает:**
- `str`: Хешированный пароль в виде hex-строки

**Пример:**
```python
>>> h("mypassword")
'a1b2c3d4...'
```

---

### `v(p: str, hx: str) -> bool`

Проверяет соответствие пароля хешу.

**Параметры:**
- `p` (str): Исходный пароль для проверки
- `hx` (str): Хеш для сравнения

**Возвращает:**
- `bool`: True если пароль совпадает с хешем

**Пример:**
```python
>>> v("mypassword", h("mypassword"))
True
```

---

### `gen_token(d: dict, exp: Optional[int] = None) -> str`

Генерирует JWT токен для пользователя.

**Параметры:**
- `d` (dict): Словарь данных для кодирования
- `exp` (Optional[int]): Время истечения в минутах

**Возвращает:**
- `str`: JWT токен в виде строки

**Пример:**
```python
>>> gen_token({"sub": "testuser"})
'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```

---

### `decode_token(t: str) -> Optional[dict]`

Декодирует JWT токен и возвращает данные.

**Параметры:**
- `t` (str): JWT токен для декодирования

**Возвращает:**
- `Optional[dict]`: Словарь с данными или None

---

### `find_user(u: str) -> Optional[dict]`

Ищет пользователя в базе по имени.

**Параметры:**
- `u` (str): Имя пользователя для поиска

**Возвращает:**
- `Optional[dict]`: Данные пользователя или None

---

### `create_user(u: str, p: str) -> dict`

Создаёт нового пользователя в системе.

**Параметры:**
- `u` (str): Имя пользователя (мин. 3 символа)
- `p` (str): Пароль (мин. 6 символов)

**Возвращает:**
- `dict`: Данные созданного пользователя

**Исключения:**
- `HTTPException`: Если данные невалидны

---

### `authenticate_user(u: str, p: str) -> Optional[str]`

Аутентифицирует пользователя по имени и паролю.

**Параметры:**
- `u` (str): Имя пользователя
- `p` (str): Пароль для проверки

**Возвращает:**
- `Optional[str]`: JWT токен или None
