## Обзор [[id:overview]]

Фотобанк поддерживает поиск по:

| Поле | Описание |
|------|----------|
| **Название** | Имя фотографии |
| **Описание** | Текст описания |
| **Теги** | Ключевые слова |
| **Автор** | Имя загрузившего |

---

## Поиск по полям [[id:fields]]

### По названию

Введите слова из названия:

```
Запрос: "закат"
Найдёт: "Закат в горах", "Красивый закат"
```

### По описанию

Поиск работает по полному тексту описания:

```
Запрос: "океан"
Найдёт: фото с "океан" в описании
```

### По тегам

Наиболее точный поиск:

```
Запрос: "горы"
Найдёт: фото с тегом "горы"
```

### По автору

Через поиск по никнейму:

```
Запрос: "admin"
Найдёт: фото от @admin
```

---

## Комбинирование [[id:combining]]

### Несколько слов

Введите несколько слов для уточнения:

```
Запрос: "горы снег"
Найдёт: фото с обоими словами
```

### Частичное совпадение

Поиск работает по частичным совпадениям:

```
Запрос: "гор"
Найдёт: "гора", "горы", "горный"
```

## Советы [[id:tips]]

### Используйте теги

Теги дают наиболее точные результаты:

| Запрос | Результат |
|--------|-----------|
| `природа` | Все фото природы |
| `закат` | Фото с закатом |

### Короткие запросы

1-2 слова работают лучше длинных фраз:

```
✅ "горы"
❌ "хочу найти фото с горами"
```

### Быстрые теги

Используйте кнопки для быстрого доступа:

```
[Природа] [Архитектура] [Люди] [Технологии] [Искусство]
```

### Поиск по автору

Для поиска фото конкретного автора:

1. Нажмите на аватар автора
2. Выберите «Фото автора»
