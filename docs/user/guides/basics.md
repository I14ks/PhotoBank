## Шаг 1: Регистрация [[id:step-1]]

Первое, что нужно сделать — создать аккаунт.

```
┌─────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]         🔐 Войти │
└─────────────────────────────────────────────┘
                    └────────┘
                        │
                        ▼
                 Нажмите сюда
```

**Действия:**

1. Нажмите **«Войти»**
2. Выберите **«Регистрация»**
3. Введите никнейм (от 3 символов)
4. Введите пароль (от 6 символов)
5. Нажмите **«Зарегистрироваться»**

**Готово!** Теперь у вас есть аккаунт.

---

## Шаг 2: Вход [[id:step-2]]

После регистрации вы автоматически входите в систему.

Для последующих входов:

1. Нажмите **«Войти»**
2. Введите никнейм и пароль
3. Нажмите **«Войти»**

**Индикатор входа:**

```
┌──────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    ⭐  📤  👤 you ▼  │
│                                    └────┘       │
│                                     │           │
│                                     ▼           │
│                              Вы вошли в систему │
└──────────────────────────────────────────────────┘
```

## Шаг 3: Поиск [[id:step-3]]

### Простой поиск

1. Введите запрос в строку поиска
2. Нажмите Enter

### Быстрый поиск

Нажмите на кнопку с тегом:

```
[Природа] [Архитектура] [Люди] [Технологии] [Искусство]
```

---

## Шаг 4: Загрузка [[id:step-4]]

### Загрузка первого фото

1. Нажмите **«Загрузить»**
2. Введите URL изображения
3. Добавьте название
4. Добавьте описание (по желанию)
5. Добавьте теги
6. Нажмите **«Опубликовать»**

**Пример:**

| Поле | Значение |
|------|----------|
| URL | `https://picsum.photos/seed/test/800/600` |
| Название | `Моё первое фото` |
| Описание | `Пробная загрузка` |
| Теги | `тест, первое` |

---

## Шаг 5: Избранное [[id:step-5]]

### Добавление в избранное

1. Найдите интересное фото
2. Нажмите на иконку **❤️**

### Просмотр избранного

1. Нажмите на иконку **⭐** в верхней панели
2. Откроется страница с избранными фото
