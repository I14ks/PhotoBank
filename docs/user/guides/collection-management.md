## Обзор [[id:overview]]

Коллекции позволяют:

- 📁 **Группировать** фото по темам
- 🔍 **Быстро находить** сохранённое
- 📤 **Делиться** подборками

---

## Создание коллекции [[id:creating]]

> **Примечание:** В текущей версии коллекции не реализованы. Используйте избранное.

### Временное решение

1. Добавляйте фото в **Избранное**
2. Используйте **теги** для организации
3. Создавайте **запросы** для фильтрации

---

## Организация [[id:organizing]]

### Через избранное

| Действие | Описание |
|----------|----------|
| **Добавить** | Нажмите ❤️ на фото |
| **Удалить** | Нажмите ❤️ ещё раз |
| **Просмотр** | Нажмите ⭐ в панели |

### Через теги

Добавляйте теги к своим фото для группировки:

```
[горы] [закат] [природа] [отпуск]
```

---

## Экспорт [[id:exporting]]

> **Примечание:** В текущей версии экспорт не реализован.

### Временное решение

1. Откройте **Избранное**
2. Скачайте нужные фото по одному
3. Сохраните в локальную папку
