## Что такое Фотобанк [[id:what-is]]

**Фотобанк** — интерактивная веб-платформа для:

- 🔍 **Визуального поиска** изображений
- 📤 **Загрузки** собственных фотографий
- ❤️ **Добавления в избранное**
- 📝 **Редактирования** метаданных
- 🌐 **Обмена** контентом с другими пользователями

---

## Регистрация [[id:registration]]

Для начала работы необходимо создать учётную запись.

### Шаги регистрации

1. Нажмите кнопку **«Войти»** в правом верхнем углу
2. Выберите вкладку **«Регистрация»**
3. Введите **никнейм** (минимум 3 символа)
4. Введите **пароль** (минимум 6 символов)
5. Нажмите **«Зарегистрироваться»**

> **Примечание:** После регистрации вы автоматически войдёте в систему.

**См.** [[wiki/registration|Подробное руководство по регистрации]]

## Первые шаги [[id:first-steps]]

После входа в систему:

| Действие | Описание |
|----------|----------|
| 🔍 **Поиск** | Введите запрос в строку поиска |
| 📤 **Загрузка** | Нажмите «Загрузить» для добавления фото |
| ❤️ **Избранное** | Нажмите на иконку закладки |
| 👤 **Профиль** | Доступен через аватар пользователя |

## Навигация [[id:navigation]]

### Главное меню

```
┌─────────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    🌐 🌙  ⭐  📤  👤 ▼  │
└─────────────────────────────────────────────────────┘
```

| Элемент | Описание |
|---------|----------|
| 📷 **Логотип** | Возврат на главную |
| 🔍 **Поиск** | Поиск по названию, описанию, тегам |
| 🌐 **Язык** | Переключение RU/EN |
| 🌙 **Тема** | Светлая/Тёмная тема |
| ⭐ **Избранное** | Просмотр избранных фото |
| 📤 **Загрузить** | Загрузка нового фото |
| 👤 **Профиль** | Меню пользователя |
