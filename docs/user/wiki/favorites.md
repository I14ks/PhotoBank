## Обзор [[id:overview]]

Избранное позволяет:

- ❤️ **Сохранять** понравившиеся фото
- 📁 **Организовать** коллекцию
- 🔍 **Быстро находить** сохранённое

```
┌──────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    ⭐ 3    📤  👤    │
│                                  └┬┘             │
│                                   │              │
│                                   ▼              │
│                         Число избранных фото     │
└──────────────────────────────────────────────────┘
```

## Добавление [[id:adding]]

### Из сетки фото

Нажмите на иконку **закладки** на карточке фото:

```
┌─────────────────────┐
│  ┌───────────────┐  │
│  │               │  │
│  │     📷        │  │
│  │   Фотография  │  │
│  │               │  │
│  └───────────────┘  │
│  ❤️ Добавить        │
│     └────┘          │
│      │              │
│      ▼              │
│   Нажмите           │
└─────────────────────┘
```

### Из просмотра фото

Откройте фото и нажмите **«В избранное»**:

```
┌─────────────────────────────────────┐
│  ┌───────────────────────────────┐  │
│  │                               │  │
│  │         Большое фото          │  │
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  Название фото                      │
│  от @author                         │
│                                     │
│  [❤️ В избранное] [⬇️ Скачать]     │
└─────────────────────────────────────┘
```

### Индикатор состояния

| Иконка | Состояние |
|--------|-----------|
| 🤍 | Не в избранном |
| ❤️ | В избранном |

## Просмотр [[id:viewing]]

### Страница избранного

1. Нажмите на иконку **⭐** в верхней панели
2. Откроется страница со всеми избранными фото

```
┌──────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    ⭐ 3    📤  👤    │
├──────────────────────────────────────────────────┤
│  ❤️ Мои избранные (3)                           │
├──────────────────────────────────────────────────┤
│  ┌───────┬───────┬───────┐                       │
│  │  📷   │  📷   │  📷   │                       │
│  │ ❤️    │ ❤️    │ ❤️    │                       │
│  └───────┴───────┴───────┘                       │
└──────────────────────────────────────────────────┘
```

### Пустое избранное

Если избранное пусто:

```
┌──────────────────────────────────────────────────┐
│  ❤️ Мои избранные                               │
├──────────────────────────────────────────────────┤
│                                                  │
│        📭 У вас пока нет избранных фото          │
│                                                  │
│        [Смотреть все фото]                       │
│                                                  │
└──────────────────────────────────────────────────┘
```

## Удаление [[id:removing]]

### Из сетки фото

Нажмите на **❤️** ещё раз:

```
┌─────────────────────┐
│  ┌───────────────┐  │
│  │     📷        │  │
│  └───────────────┘  │
│  ❤️ В избранном     │
│  └────┘             │
│   │                 │
│   ▼                 │
│  Нажмите для        │
│  удаления           │
└─────────────────────┘
```

### Из страницы избранного

1. Откройте **Избранное**
2. Нажмите **❤️** на фото
3. Фото удалится из коллекции
