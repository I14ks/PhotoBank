## Требования к аккаунту [[id:requirements]]

| Параметр | Требование |
|----------|------------|
| **Никнейм** | Минимум 3 символа |
| **Пароль** | Минимум 6 символов |
| **Уникальность** | Никнейм должен быть уникален |

> **Важно:** В системе используется хеширование паролей для безопасности.

## Процесс регистрации [[id:process]]

### Шаг 1: Открытие формы

Нажмите кнопку **«Войти»** в правом верхнем углу страницы.

```
┌────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    🔐 Войти │
└────────────────────────────────────────┘
           └───────────────┘
                  │
                  ▼
           Нажмите здесь
```

### Шаг 2: Переключение на регистрацию

В открывшемся окне выберите вкладку **«Регистрация»**.

### Шаг 3: Ввод данных

| Поле | Значение | Пример |
|------|----------|--------|
| **Никнейм** | Ваше имя пользователя | `photo_master` |
| **Пароль** | Секретный код | `mysecret123` |

### Шаг 4: Подтверждение

Нажмите кнопку **«Зарегистрироваться»**.

```
┌─────────────────────────────────────┐
│  Регистрация                        │
├─────────────────────────────────────┤
│  Никнейм:  [________________]       │
│  Пароль:   [________________]       │
│                                     │
│  [Зарегистрироваться]               │
│                                     │
│  Нет аккаунта? ← Вы здесь           │
└─────────────────────────────────────┘
```

## Вход в систему [[id:login]]

### Для зарегистрированных пользователей

1. Нажмите **«Войти»**
2. Введите **никнейм** и **пароль**
3. Нажмите **«Войти»**

### После входа

- Отобразится ваш **аватар** с никнеймом
- Станет доступна **загрузка фото**
- Откроется доступ к **избранному**

```
┌─────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    ⭐  📤  P photo_master ▼ │
└─────────────────────────────────────────────────┘
                                    └────┘
                                     │
                         Вы вошли как photo_master
```

## Восстановление доступа [[id:recovery]]

> **Примечание:** В текущей версии восстановление пароля не реализовано. При утере пароля необходимо зарегистрироваться заново.

## Ошибки и решения [[id:troubleshooting]]

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Username already exists` | Никнейм занят | Выберите другой никнейм |
| `Username too short` | Менее 3 символов | Увеличьте длину никнейма |
| `Password too short` | Менее 6 символов | Придумайте более сложный пароль |
