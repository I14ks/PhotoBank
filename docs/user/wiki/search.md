## Обзор [[id:overview]]

Поиск работает по следующим полям:

| Поле | Описание |
|------|----------|
| **Название** | Поиск по названию фото |
| **Описание** | Поиск по описанию |
| **Теги** | Поиск по тегам |

```
┌──────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [🔍 Поиск вдохновения...]    ⭐  │
│                   └────────────┘                 │
│                        │                         │
│                        ▼                         │
│              Введите запрос здесь                │
└──────────────────────────────────────────────────┘
```

## Поисковый запрос [[id:query]]

### Простой поиск

Введите ключевые слова в строку поиска:

| Запрос | Результат |
|--------|-----------|
| `горы` | Фото с «горы» в названии, описании или тегах |
| `закат` | Фото с «закат» в любом поле |
| `океан` | Фото с «океан» в любом поле |

### Особенности поиска

- **Регистронезависимый** — `Горы` = `горы` = `ГОРЫ`
- **Частичное совпадение** — `гор` найдёт `горы`, `горный`
- **По всем полям** — ищет в названии, описании и тегах

## Фильтры [[id:filters]]

### Быстрые теги

На главной странице доступны кнопки для быстрого поиска:

```
┌──────────────────────────────────────────────────┐
│  [Природа] [Архитектура] [Люди] [Технологии]    │
│  [Искусство]                                     │
└──────────────────────────────────────────────────┘
```

| Тег | Описание |
|-----|----------|
| **Природа** | Пейзажи, растения, животные |
| **Архитектура** | Здания, города, сооружения |
| **Люди** | Портреты, мероприятия |
| **Технологии** | Устройства, техника |
| **Искусство** | Картины, дизайн |

### Фильтр «Мои фото»

Для просмотра только ваших загрузок:

1. Нажмите на **аватар**
2. Выберите **«Мои фото»**

## Результаты поиска [[id:results]]

### Отображение

Результаты показываются в виде **сетки карточек**:

```
┌─────────┬─────────┬─────────┐
│  📷     │  📷     │  📷     │
│  Фото 1 │  Фото 2 │  Фото 3 │
│  ❤️ 12  │  ❤️ 8   │  ❤️ 24  │
└─────────┴─────────┴─────────┘
```

### Сортировка

- **По умолчанию** — новые сверху
- **По популярности** — по количеству лайков

## Советы по поиску [[id:tips]]

| Совет | Описание |
|-------|----------|
| **Используйте теги** | Теги дают наиболее точные результаты |
| **Короткие запросы** | 1-2 слова работают лучше длинных |
| **Быстрые теги** | Используйте кнопки для быстрого доступа |
