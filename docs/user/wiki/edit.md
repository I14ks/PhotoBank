## Обзор [[id:overview]]

Редактирование позволяет изменить:

| Поле | Описание |
|------|----------|
| **Название** | Имя фотографии |
| **Описание** | Текст описания |
| **Теги** | Ключевые слова |

> **Важно:** Изменять можно только **свои** фотографии.

---

## Доступ [[id:access]]

### Из сетки фото

1. Найдите своё фото
2. Нажмите на фото для открытия
3. Выберите **«Редактировать»**

```
┌─────────────────────────────────────┐
│  ┌───────────────────────────────┐  │
│  │         Ваше фото             │  │
│  └───────────────────────────────┘  │
│                                     │
│  Название                           │
│  от @you                            │
│                                     │
│  [✏️ Редактировать] [❤️] [⬇️]      │
│    └────────┘                       │
│         │                           │
│         ▼                           │
│    Нажмите здесь                    │
└─────────────────────────────────────┘
```

### Из модального окна

1. Откройте фото
2. Нажмите **«Редактировать»** в нижней панели

## Изменение данных [[id:changing]]

### Форма редактирования

```
┌─────────────────────────────────────────────┐
│  Редактирование фото                        │
├─────────────────────────────────────────────┤
│  Название: [Закат в горах________]          │
│  Описание: [Красивый закат над...]          │
│  Теги: [горы] [закат] [природа] [+]         │
│                                             │
│  [Отмена]          [Сохранить изменения]    │
└─────────────────────────────────────────────┘
```

### Поля

| Поле | Можно изменить |
|------|----------------|
| **Название** | ✅ Да |
| **Описание** | ✅ Да |
| **Теги** | ✅ Да |
| **URL** | ❌ Нет |
| **Автор** | ❌ Нет |

---

## Сохранение [[id:saving]]

### Шаги

1. Внесите изменения в поля
2. Нажмите **«Сохранить изменения»**
3. Дождитесь подтверждения

### Индикатор сохранения

```
┌─────────────────────────────────────────────┐
│  Редактирование фото                        │
├─────────────────────────────────────────────┤
│  ...                                        │
│  [Отмена]          [⏳ Сохранение...]       │
└─────────────────────────────────────────────┘
```

### Успешное сохранение

- Модальное окно закроется
- Изменения отобразятся сразу

---

## Ограничения [[id:limitations]]

| Ограничение | Описание |
|-------------|----------|
| **Только свои фото** | Нельзя чужие |
| **Требуется вход** | Нужен аккаунт |
| **URL неизменен** | Ссылку нельзя поменять |

---

## Ошибки и решения [[id:troubleshooting]]

| Проблема | Причина | Решение |
|----------|---------|---------|
| `Not authorized` | Не ваше фото | Можно только свои |
| `Not found` | Фото удалено | Обновите страницу |
| `Unauthorized` | Не вошли | Войдите в аккаунт |
