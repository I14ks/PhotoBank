## Обзор [[id:overview]]

Фотобанк позволяет скачивать изображения для:

- 📱 **Использования** в личных целях
- 💾 **Сохранения** на устройстве
- 📤 **Поделиться** с другими

> **Примечание:** Проверяйте лицензию перед использованием!

---

## Процесс [[id:process]]

### Шаг 1: Открытие фото

Нажмите на изображение для просмотра:

```
┌─────────────────────────────────────┐
│  ┌───────────────────────────────┐  │
│  │                               │  │
│  │         Фотография            │  │
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  Название фото                      │
│  от @author                         │
│                                     │
│  [❤️] [⬇️ Скачать] [✏️]            │
│       └────┘                        │
│         │                           │
│         ▼                           │
│    Нажмите здесь                    │
└─────────────────────────────────────┘
```

### Шаг 2: Скачивание

Нажмите кнопку **«Скачать»**:

```
┌─────────────────────────────────────┐
│  ┌───────────────────────────────┐  │
│  │         Фотография            │  │
│  └───────────────────────────────┘  │
│                                     │
│  ⬇️ Загрузка...                     │
│                                     │
└─────────────────────────────────────┘
```

### Шаг 3: Сохранение

Файл сохранится в папку **Загрузки** по умолчанию.

## Форматы [[id:formats]]

### Доступные форматы

| Формат | Описание |
|--------|----------|
| **Оригинал** | Исходное качество |
| **Web** | Оптимизировано для веба |

### Определение формата

Формат зависит от источника изображения:

| Источник | Формат |
|----------|--------|
| Picsum | JPG |
| Unsplash | JPG/PNG |
| Прямая ссылка | Зависит от URL |

## Ошибки и решения [[id:troubleshooting]]

| Проблема | Причина | Решение |
|----------|---------|---------|
| `Не скачивается` | Блокировка браузера | Разрешите скачивание |
| `Пустой файл` | Ошибка URL | Проверьте ссылку |
| `404` | Фото удалено | Обратитесь к автору |
