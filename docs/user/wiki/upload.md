## Требования к фото [[id:requirements]]

| Параметр | Требование |
|----------|------------|
| **Формат** | Любой доступный по URL |
| **URL** | Должен быть доступен из интернета |
| **Название** | Обязательно (минимум 1 символ) |
| **Описание** | Опционально |
| **Теги** | Опционально (рекомендуется) |

## Процесс загрузки [[id:process]]

### Шаг 1: Открытие формы

Нажмите кнопку **«Загрузить»** в верхней панели.

```
┌──────────────────────────────────────────────────┐
│  📷 ФОТОБАНК    [Поиск...]    ⭐  📤 Загрузить   │
│                                  └────┘          │
│                                   │              │
│                                   ▼              │
│                            Нажмите здесь         │
└──────────────────────────────────────────────────┘
```

### Шаг 2: Заполнение формы

| Поле | Описание | Пример |
|------|----------|--------|
| **URL** | Ссылка на изображение | `https://example.com/photo.jpg` |
| **Название** | Имя фотографии | `Закат в горах` |
| **Описание** | История фото | `Красивый закат...` |
| **Теги** | Ключевые слова | `горы, закат, природа` |

```
┌─────────────────────────────────────────────┐
│  Загрузка фото                              │
├─────────────────────────────────────────────┤
│  URL: [https://example.com/photo.jpg]       │
│  Название: [Закат в горах]                  │
│  Описание: [Красивый закат над горами...]   │
│  Теги: [горы] [закат] [природа] [+]         │
│                                             │
│  [Отмена]          [Опубликовать]           │
└─────────────────────────────────────────────┘
```

### Шаг 3: Публикация

Нажмите **«Опубликовать»** для завершения загрузки.

## Метаданные [[id:metadata]]

### Название

- **Обязательное поле**
- Отображается в карточке фото
- Используется при поиске

### Описание

- **Опциональное поле**
- Рекомендуется для лучшего понимания
- Отображается при просмотре

### Теги

- **Опциональное поле**
- Улучшает поиск по фото
- Рекомендуется добавлять 3-5 тегов

**Популярные теги:**

| Тег | Описание |
|-----|----------|
| `природа` | Пейзажи, растения, животные |
| `архитектура` | Здания, сооружения |
| `люди` | Портреты, события |
| `технологии` | Гаджеты, устройства |
| `искусство` | Картины, скульптуры |

## Управление загрузками [[id:management]]

### Просмотр своих фото

1. Нажмите на **аватар** в верхней панели
2. Выберите **«Мои фото»**

### Редактирование

1. Откройте нужное фото
2. Нажмите **«Редактировать»**
3. Измените метаданные
4. Нажмите **«Сохранить»**

### Удаление

1. Откройте нужное фото
2. Нажмите **«Удалить»**
3. Подтвердите действие

> **Важно:** Можно удалять только свои фотографии.

## Ошибки и решения [[id:troubleshooting]]

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Invalid URL` | Недоступная ссылка | Проверьте URL |
| `Title required` | Пустое название | Введите название |
| `Unauthorized` | Не вошли в систему | Войдите в аккаунт |
