## Как загрузить [[id:upload]]

**Шаги:**

1. Войдите в аккаунт
2. Нажмите **«Загрузить»**
3. Введите URL изображения
4. Заполните название и описание
5. Добавьте теги
6. Нажмите **«Опубликовать»**

**См.** [[wiki/upload|Подробное руководство]]

---

## Как редактировать [[id:edit]]

**Шаги:**

1. Откройте своё фото
2. Нажмите **«Редактировать»**
3. Измените поля
4. Нажмите **«Сохранить»**

**Можно изменить:**

| Поле | Можно |
|------|-------|
| Название | ✅ |
| Описание | ✅ |
| Теги | ✅ |
| URL | ❌ |

**См.** [[wiki/edit|Редактирование]]

---

## Как удалить [[id:delete]]

**Шаги:**

1. Откройте своё фото
2. Нажмите **«Удалить»**
3. Подтвердите действие

> **Важно:** Можно удалять только свои фото!

## Как найти [[id:find]]

### Поиск по сайту

1. Введите запрос в **строку поиска**
2. Нажмите Enter

### Быстрые теги

Нажмите на кнопку с нужным тегом:

```
[Природа] [Архитектура] [Люди] [Технологии] [Искусство]
```

### Мои фото

1. Нажмите на **аватар**
2. Выберите **«Мои фото»**

**См.** [[wiki/search|Поиск]]

## Форматы [[id:formats]]

### Поддерживаемые форматы

| Формат | Расширение |
|--------|------------|
| JPEG | `.jpg`, `.jpeg` |
| PNG | `.png` |
| WebP | `.webp` |
| GIF | `.gif` |

### Источники

| Источник | Описание |
|----------|----------|
| **Picsum** | Случайные фото (демо) |
| **Unsplash** | Бесплатные стоковые |
| **Прямой URL** | Любое доступное фото |

## Проблемы с фото [[id:issues]]

| Проблема | Решение |
|----------|---------|
| `Не загружается` | Проверьте URL |
| `Не редактируется` | Это не ваше фото |
| `Не удаляется` | Это не ваше фото |
| `Пустая картинка` | URL недоступен |
